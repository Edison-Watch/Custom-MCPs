


## How to contribute

1. Clone repo and create a new branch: `$ git clone https://github.com/Edison-Watch/Custom-MCPs.git && cd Custom-MCPs && git checkout -b your-branch-name`
2. Make changes and test
3. Submit a PR (Pull Request) with comprehensive description of changes
