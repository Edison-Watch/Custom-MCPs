## greet

Greet someone by name.

```bash
edisonmcps greet Alice
# Hello, Alice!

edisonmcps greet Alice --shout
# HELLO, ALICE!

edisonmcps greet Bob --shout --times 3
```

| Option | Short | Default | Description |
|--------|-------|---------|-------------|
| `--shout` | `-s` | `false` | Uppercase the greeting |
| `--times` | `-t` | `1` | Repetition count |

## config

Inspect and modify configuration.

```bash
edisonmcps config show                           # view full config
edisonmcps config get llm_config.cache_enabled   # get a single value
edisonmcps config set logging.verbose false      # set an override
```

| Subcommand | Description |
|------------|-------------|
| `show` | Display the full configuration |
| `get <key>` | Get a single value by dot-separated key |
| `set <key> <value>` | Write an override to `.global_config.yaml` |

## doctor

Run health checks on the project environment.

```bash
edisonmcps doctor          # check everything
edisonmcps doctor --fix    # auto-fix what it can
```

Checks Python version, `uv` installation, dependency sync, config validity, `.env` file, pre-commit hooks, and git status.

## init

Scaffold a new command from a built-in template.

```bash
edisonmcps init my_command --desc "Does something useful"
# Creates commands/my_command.py
# Run it: edisonmcps my-command
```

| Option | Description |
|--------|-------------|
| `--desc` | Description for the new command |

The name must be `snake_case`. Underscores are converted to hyphens for the CLI command name.

## completions

Install shell tab-completions for faster usage.

```bash
edisonmcps completions install bash    # append to ~/.bashrc
edisonmcps completions install zsh     # append to ~/.zshrc
edisonmcps completions install fish    # append to config.fish
edisonmcps completions show bash       # print script to stdout
```

## update

Check for a newer version and upgrade.

```bash
edisonmcps update
```

## telemetry

Manage anonymous usage telemetry.

```bash
edisonmcps telemetry status     # show enabled/disabled
edisonmcps telemetry disable    # opt out
edisonmcps telemetry enable     # opt back in
```

You can also set `CLI_TELEMETRY_DISABLED=1` to disable.
